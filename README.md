# TASK MANAGEMENT API

A simple CRUD task management API to create tasks, show tasks, delete tasks, edit tasks, mark tasks as completed and stores all the tasks inside a sqlite3 database

# Note
This api was hosted on Render. The link is https://task-managementapi.onrender.com

##  API Endpoints

| Method | Endpoint       | Description       |
|--------|--------------- |-------------------|
| GET    | /tasks         | Get all tasks     |
| GET    | /tasks/{id}    |Get a specific task|
| POST   | /tasks         | Create a new task |
| PUT    | /tasks/{id}    | Update a task     |
| PATCH  | /tasks/{id}    | Mark as completed |
| DELETE | /tasks/{id}    | Delete a task     |
